Functionality

Adding new URLs

You can add new URLs to the blacklist either from the current open tab or from the options page. To open the extension in the current tab, click the extension icon in your browser:

You can also open the extension by using Ctrl+Q key shortcut.

Edit/delete URLs

You can edit URLs from the options page. If you try to update a URL to one that already exists, the extension will save only one instance of it.

You can delete an individual URL or delete all URLs at once. Be careful, as this action is irreversible.

Subdomains block

By toggling the "Block Subdomains" checkbox, you can choose whether to block all subdomains of a URL or just the specified one.

For example:

URL: "www.example.com",

Block subdomains: ON,

Result: blocks "www.example.com", "www.example.com/abc", "www.example.com/abc/123", etc.

URL: "www.example.com",

Block subdomains: OFF,

Result: blocks "www.example.com", but allows "www.example.com/abc", "www.example.com/abc/123", etc.

Disabling rules

If you don't want to delete a URL but still need access to the website, you can disable it by toggling the "Active" switch on the options page. This temporarily bypasses the blacklist without deleting the URL. You can re-enable it later if needed.

Important: Disabling rules behaves differently when "Strict Mode" is enabled.

Strict Mode

"Strict Mode" is designed for users who struggle to focus, get easily distracted, or have ADHD. This feature adds an additional layer of restriction by enforcing time-based reactivation. If you disable a URL with Strict Mode enabled, it will be automatically re-enabled after one hour.

FAQ

The extension doesn't work in incognito mode. How can I fix it?

Ensure the extension is allowed to run in incognito mode. To enable this:

  1. Go to your browser settings.
  2. Navigate to Extensions, then Manage Extensions.
  3. Locate Website Blocker and click Details.
  4. Turn on the Allow in Incognito Mode switch.
