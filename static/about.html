Functionality

Adding new URLs

You can add new URLs to the blacklist either from the current open tab or from the options page. To open the extension in the current tab, click the extension icon in your browser.

You can also open the extension by using Ctrl+Q key shortcut

Edit/delete URLs

You can edit URLs in the options page. If you attempt to update URL with the an already existing URL, the extension will only save one.

You can delete an individual URL or delete all URLs. Be careful as this action is unrevertable.

Subdomains block

By toggling block subdomains checkbox you you can decide for each URL whether to block all the subdomains of the URL, or to just block the specified one.

For example:

URL: "www.example.com",

Block subdomains: ON,

Result: will block "www.example.com", "www.example.com/abc", "www.example.com/abc/123", etc.

URL: "www.example.com",

Block subdomains: OFF,

Result: will block "www.example.com", but allow "www.example.com/abc", "www.example.com/abc/123", etc.

Disabling rules

If you don't want to delete an URL, but still need to get access to that website, you can disable it by turning of the "Active" switch on options page. This will allow the website to bypass the extension without deleting it from the blacklist. You can enable it back when you need to restict it again.

Important! Disabling rules behaves differently when "Strict mode" is on.

Strict Mode

"Strict mode" is a feature made for users who experience difficulties focusing on work and get easily distracted and those suffering from ADHD. Strict mode adds an additional layer restriction by enforcing URLs enabling. If you disable URL with strict mode on, it will be automatically enabled back in 1 hour.

FAQ

The extension doesn't work in incognito mode. How to fix it?

Make sure that you have allowed to use the extension to work in incognito mode. To do that go to browser -> Extensions -> Extensions settings -> Website Blocker -> turn on the "Allow in incognito mode" switch
